🎓 Alumni Platform for University

A secure web-based alumni networking platform that connects students, alumni, and faculty of a university.
The system allows verified users to interact, share updates, build connections, and stay informed about events and opportunities.

🚀 Project Overview

This platform is designed to create a private university community where only authorized users can access the system.
Students, alumni, and teachers can connect, communicate, and share updates similar to a social networking platform.

The website is built using PHP with a focus on security, controlled access, and community engagement.

🔐 Secure Login System

Only users added by admin can log in

Admin enters student/alumni data in dashboard

Unauthorized users cannot create accounts

Ensures a private and trusted network
